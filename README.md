# QA Assignment Submission

This repository contains all deliverables for the QA assessment.

## Repository Structure

- `bug-reports/` → Bug reports and defect documentation
- `test-cases/` → Manual test cases
- `playwright-automation/` → Playwright automation framework using TypeScript

## Automation Coverage

The Playwright automation includes:
- Login Test
- Add To Cart Test
- Checkout Test

## Tech Used

- Playwright
- TypeScript
- Node.js
