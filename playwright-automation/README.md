# Playwright Automation Assignment

This project contains automated test scenarios for the SauceDemo website using Playwright with TypeScript.

## Scenarios Covered

- Successful Login
- Add Product to Cart
- Successful Checkout

## Tech Stack

- Playwright
- TypeScript
- Node.js

## Framework Design

The project follows the Page Object Model (POM) design pattern for better readability and reusability.

## Run The Tests

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npx playwright test
```

Open HTML report:

```bash
npx playwright show-report
```

## Features

- Reusable page objects
- Assertions for validations
- HTML reporting
- Screenshots/videos on failure
- Cross-browser capable structure
